# Titanic Dataset - Exploratory Data Analysis (EDA)

## About
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to find hidden patterns and insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- Most passengers were aged between 20-40.
- First class passengers had higher survival rates.
- Higher ticket fares were associated with better survival chances.
- Age, Pclass, and Fare show correlation with survival.

## How to Run
1. Place the `train.csv` file in your working directory.
2. Run the `Titanic_EDA.ipynb` notebook.
3. Visualizations and insights will be displayed.
